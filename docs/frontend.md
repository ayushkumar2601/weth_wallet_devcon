# WETH Frontend

The WETH Frontend is a Next.js 15 App Router application providing a secure, read-only interface for human intervention in AI-generated transactions.

## Tech Stack
- Next.js 15
- React 19
- Tailwind CSS v4
- Wagmi & RainbowKit (Wallet connections)
- React Query (Data fetching)

## Pages
1. `/dashboard`: Overview of ETH, Tokens, and Pending Transactions.
2. `/portfolio`: In-depth breakdown of all ERC20 holdings.
3. `/transactions`: Historical transfers mapping.
4. `/risk`: Suspicious approval detector and Portfolio heuristics.
5. `/audit`: Global system execution trace for compliance.
