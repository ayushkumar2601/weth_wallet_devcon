# Roadmap

## Phase 1 (Completed)
- Monorepo setup with pnpm.
- Fastify REST API.
- MCP Server integration.
- Read-only queries to Sepolia (balance, ENS).
- Database and Redis integration.

## Phase 2 (Upcoming)
- Write operations (transaction signing).
- Indexer integration (Alchemy) for historical txs and tokens.
- DeFi swaps, staking, bridging integrations.
- Support for Mainnet, Optimism, Arbitrum, Base.
