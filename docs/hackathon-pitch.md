# WETH: Hackathon Pitch

**Tagline:** AI on Ethereum, Without the Private Key Risk.

## The Problem
Everyone wants AI to execute transactions on-chain. But giving an LLM a private key is a massive security risk. Custodial AI agents are dangerous, uninsurable, and a single point of failure.

## The Solution: WETH
WETH uses the Model Context Protocol (MCP) to let Claude draft, estimate, and simulate transactions natively in its chat window. But the execution stops there.
WETH implements a strict **Human-in-the-Loop** model. The AI drafts the transaction to a database. The human logs into a sleek Web3 Dashboard, reviews the Risk/Policy score, and signs it using their own hardware or browser wallet.

## Why We Win
1. **Zero-Trust**: We never hold private keys.
2. **AI Native**: Integrated natively into Claude Desktop using standard I/O.
3. **Enterprise Grade Security**: Every drafted transaction is dry-run simulated on Sepolia, risk-scored, and auditable.
